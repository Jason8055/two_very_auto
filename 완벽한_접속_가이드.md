## 🛠️ 문제 해결

### ❌ "사이트에 연결할 수 없음" 오류 시

#### 해결책 1: 새 배치 파일 사용
```
완벽한_페어시스템_시작.bat 실행
```
- 이 파일이 모든 문제를 자동으로 해결합니다

#### 해결책 2: 수동 방화벽 설정
```cmd
netsh advfirewall firewall add rule name="TwoVeryAuto" dir=in action=allow protocol=TCP localport=8080
```

#### 해결책 3: 다른 포트 확인
- 브라우저에서 http://127.0.0.1:8000 시도
- 브라우저에서 http://127.0.0.1:3000 시도

### ❌ 서버 시작 실패 시

#### 해결책 1: 프로세스 완전 정리
```cmd
taskkill /f /im python.exe
```
그 후 배치 파일 다시 실행

#### 해결책 2: 관리자 권한으로 실행
- 배치 파일에서 마우스 우클릭 → "관리자 권한으로 실행"

## 🎰 페어 감지 시스템 사용법

### 📊 페어 대시보드 기능
1. **실시간 페이지 감지**: 플레이어/뱅커 페어 자동 감지
2. **통계 표시**: 총 페어 수, 타입별 통계
3. **상세 정보**: 게임 번호, 승부 결과, 점수
4. **자동 새로고침**: 30초마다 자동 업데이트

### 🔄 실시간 업데이트
- WebSocket 연결로 즉시 알림
- 새로운 페어 감지 시 자동 화면 업데이트
- 브라우저를 새로고침할 필요 없음

## 🚨 응급 상황 대응

### 서버가 갑자기 중단된 경우
1. **자동 복구**: 완벽한 시작 스크립트가 자동으로 재시작
2. **수동 복구**: 배치 파일을 다시 더블클릭
3. **브라우저**: F5키로 새로고침

### 포트 충돌이 계속 발생하는 경우
1. **자동 해결**: 스크립트가 자동으로 다른 포트 사용
2. **확인 방법**: 콘솔 창에서 "선택된 포트: XXXX" 메시지 확인
3. **브라우저**: http://127.0.0.1:XXXX 로 접속

## 🎉 성공 확인 방법

### ✅ 서버 정상 작동 확인

3. **자동 새로고침**: "마지막 업데이트" 시간이 갱신됨
