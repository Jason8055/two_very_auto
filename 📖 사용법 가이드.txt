🎯 Two Very Auto - 원클릭 서버 실행 가이드
===============================================

🎉 이제 버튼 하나만 누르면 모든 서버가 자동으로 실행됩니다!

📋 실행 방법 (4가지 중 선택)
===============================================

1. 🏆 【가장 쉬운 방법】 바탕화면 바로가기
   ┌─────────────────────────────────────────┐
   │ ① "바탕화면_바로가기_생성.bat" 실행     │
   │ ② 바탕화면에 생긴 아이콘 더블클릭       │
   │ ③ 끝! 자동으로 브라우저가 열립니다      │
   └─────────────────────────────────────────┘

2. 🚀 【빠른 실행】 배치 파일
   ┌─────────────────────────────────────────┐
   │ "🚀 서버 시작.bat" 더블클릭             │
   │ → 자세한 로그와 함께 서버 실행          │
   └─────────────────────────────────────────┘

3. ⚡ 【초고속 실행】 간단 버전
   ┌─────────────────────────────────────────┐
   │ "빠른_서버_시작.bat" 더블클릭           │
   │ → 최소 메시지로 빠른 실행               │
   └─────────────────────────────────────────┘

4. 🎮 【GUI 런처】 그래픽 인터페이스
   ┌─────────────────────────────────────────┐
   │ "launcher.py" 실행 또는                │
   │ 바탕화면 "Two Very Auto GUI" 아이콘     │
   │ → 예쁜 그래픽 화면에서 서버 관리        │
   └─────────────────────────────────────────┘

🌐 접속 방법
===============================================
서버가 실행되면 자동으로 브라우저가 열립니다!

수동으로 접속하려면:
• 메인 페이지: http://127.0.0.1:8000
• 페어 대시보드: http://127.0.0.1:8000/pair-dashboard
• 종합 통계: http://127.0.0.1:8000/comprehensive-dashboard

🛑 서버 종료 방법
===============================================
• 배치 파일 실행 시: Ctrl + C
• GUI 런처 실행 시: "서버 종료" 버튼 또는 창 닫기
• 명령창에서: Ctrl + C

⚠️ 문제 해결
===============================================

Q: Python이 없다는 오류가 나타나요
A: https://python.org 에서 Python 설치 후 재시도

Q: 포트가 이미 사용 중이라고 나와요  
A: 자동으로 다른 포트(8080, 3000 등)를 찾아 실행됩니다

Q: 브라우저가 자동으로 안 열려요
A: 수동으로 http://127.0.0.1:8000 에 접속하세요

Q: 서버가 시작되지 않아요
A: 관리자 권한으로 실행하거나 방화벽 설정을 확인하세요
