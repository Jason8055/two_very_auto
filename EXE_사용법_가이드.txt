🎉 Two Very Auto - EXE 파일 생성 완료!

===============================================================
                     🚀 생성된 EXE 파일
===============================================================

✅ TwoVeryAuto_프로런처.exe (GUI 버전 - 권장)
   └─ 🎮 예쁜 GUI 화면으로 서버 실행
   └─ 🎯 아이콘, 버전 정보 포함
   └─ 📱 상태 표시 및 실시간 로그
   └─ 3가지 실행 옵션 제공

✅ TwoVeryAuto_원클릭실행.exe (콘솔 버전)
   └─ ⚡ 즉시 BAT 파일 실행
   └─ 📟 콘솔 화면으로 진행 상황 표시
   └─ 🔧 오류 시 자세한 안내 제공

===============================================================
                      🚀 사용법 안내
===============================================================

🏆 프로런처 (권장):
   └─ TwoVeryAuto_프로런처.exe 더블클릭
   └─ 예쁜 화면에서 원하는 실행 방법 선택
   └─ 자동으로 브라우저 열림

⚡ 원클릭실행:
   └─ TwoVeryAuto_원클릭실행.exe 더블클릭  
   └─ 바로 BAT 파일 실행
   └─ 가장 빠른 실행 방법

===============================================================
                      💡 추가 팁
===============================================================

🔗 바탕화면에 바로가기 생성
🔗 작업표시줄에 고정하여 빠른 접근
🔗 시작 프로그램에 등록하여 부팅시 자동 실행

🌐 서버 주소: http://127.0.0.1:8000
📊 페어 대시보드: http://127.0.0.1:8000/pair-dashboard
